The toFixed() method converts a number to a string and rounds the string to a specified number of decimals.

If the number of decimals is not specified, the string is rounded to the nearest whole number.

If the number of specified decimals is larger than the decimals in the number, trailing zeros are added to make up the difference.


The valueOf() method returns the primitive value of a data type.

That is to say, it returns data that is not an object.

It returns the value of the object itself but does not change the original object.

It would appear that there is a valueOf() method for any object.
